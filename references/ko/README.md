# 프로덕트 레드 그로스 플레이북 — 프리미엄에서 엔터프라이즈 전환까지

제품이 스스로 영업하게 하세요. 이 플레이북은 PLG 모션 전체를 다룹니다.

## 프리미엄 설계

무엇을 게이트하고, 무엇을 무료로 제공하며, 가격 심리학을 어떻게 활용할 것인가.

- **가치 지표**: 고객이 자연스럽게 세는 단위를 식별 (시트, 프로젝트, 스토리지, API 호출)
- **무료 티어 범위**: "아하 모먼트"에 도달하기에 충분하지만 비즈니스를 운영하기엔 부족한 범위
- **업그레이드 트리거**: 실제 가치를 얻는 것과 상관관계가 있는 사용 한도 (임의적인 벽이 아닌)
- **가격 심리학**: 높게 앵커링, 절약 보여주기, 중간 티어를 명확한 선택으로 만들기

## 셀프서브 온보딩

관심을 잃기 전에 사용자를 활성화하기 위해 타임 투 밸류를 단축합니다.

- **첫 세션 목표**: "활성화됨"의 정의 (첫 X 생성, 팀원 초대, 설정 완료)
- **점진적 공개**: 모든 것을 한번에 보여주지 않음 — 핵심 액션을 순서대로 안내
- **빈 상태**: 모든 빈 화면은 교육과 행동 촉진의 기회
- **마일스톤 축하**: 커밋먼트를 강화하기 위해 진척도를 시각적으로 표시

## 바이럴 메카닉스

빌트인 공유, 협업 훅, 네트워크 효과.

- **설계에 의한 협업**: 여러 사용자가 필요한 제품은 더 빨리 확산
- **공유 가능한 산출물**: 자연스럽게 공유되는 결과물 (리포트, 링크, 임베드)
- **리퍼럴 프로그램**: 추천인과 피추천인 모두에게 프리미엄 기능으로 보상
- **네트워크 효과**: 새 사용자가 늘어날수록 기존 사용자에게 더 큰 가치 제공

## 사용량 기반 시그널

확장 준비가 된 어카운트를 자동으로 식별합니다.

- **PQL 정의**: Product Qualified Lead = 사용량 임계값을 지속적으로 초과하는 어카운트
- **시그널 스코어링**: 인텐트 레벨로 액션에 가중치 부여 (팀원 초대 > 문서 편집 > 대시보드 조회)
- **자동 알림**: 어카운트가 PQL 임계값을 넘을 때 세일즈에 알림
- **타이밍**: 트라이얼 만료 시가 아닌, 가치가 증명되었을 때 접근

## 엔터프라이즈 업셀

언제, 어떻게 PLG 모션에 세일즈를 추가할 것인가.

- **바텀업 채택**: 개인 사용자→팀 채택→부서→엔터프라이즈 계약
- **엔터프라이즈 트리거**: 5명 이상 활성 시트, 컴플라이언스 요구, SSO 요청, API 사용량 급증
- **세일즈 어시스트 모델**: 세일즈는 게이트키핑이 아닌 가치를 추가 (커스텀 온보딩, 보안 리뷰)
- **계약 확장**: 팀으로 진입하여 성공 사례를 통해 조직 전체로 확대
