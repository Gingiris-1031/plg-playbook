# 🪄 Product-Led Growth Playbook — Freemium to Enterprise

Let the product do the selling. This playbook maps the full PLG motion: **freemium design** (what to gate, what to give away, pricing psychology), **self-serve onboarding** (time-to-value and activation milestones), **built-in viral mechanics** (sharing hooks, collaboration loops, network effects), and **usage-based expansion signals** that tell you which accounts are enterprise-ready before they tell you.

> 📌 Canonical source: **[gingiris-skills/skills/plg-playbook](https://github.com/Gingiris-1031/gingiris-skills/tree/main/skills/plg-playbook)** — this repo is a distribution mirror. The PLG-vs-SLG decision itself lives in [gingiris-b2b-growth](https://github.com/Gingiris-1031/gingiris-b2b-growth).
